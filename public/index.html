* 입력하시면 준원이가 대사 중에 사용자님의 이름을 부르며 말을 걸어 훨씬 더 몰입감이 높아집니다!

📂 실제 준원이 카톡 대화 업로드 (선택사항)

💡 이게 무엇인가요?
준원이와 실제 나눈 카카오톡 대화 내보내기(.txt) 파일을 여기에 넣으시면, AI가 준원이의 실제 말투(자주 쓰는 비속어, 독특한 어조, 타자 습관 등)를 동적으로 반영하여 훨씬 더 사실적이고 재미있는 맞춤형 시나리오를 자동 생성해 줍니다! (파일은 브라우저 로컬에서만 임시 분석되며 개인정보는 유출되지 않습니다.)

📌 추출 방법: 카카오톡 앱 → 준원이와 대화방 → 우측 상단 메뉴(≡) → 설정(톱니바퀴) → '대화 내보내기' → 텍스트 파일 저장
